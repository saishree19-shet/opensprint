## About OpenSprint

OpenSprint is an open-source contribution event organized by the Sahyadri Open Source Community (SOSC) at Sahyadri College of Engineering and Management.

The event encourages students to explore open-source development, collaborate with developers, and contribute to real-world projects through GitHub by working on issues, improving documentation, fixing bugs, and submitting pull requests.

---

## My Participation

During OpenSprint 2026, I explored beginner-friendly issues and learned the workflow of contributing to open-source projects using GitHub.

* Understanding open-source contribution workflow
* Exploring repositories and issues
* Learning how to fork repositories
* Creating branches and making commits
